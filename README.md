# System Error Alert

🚧 Work in progress

A Linux system monitoring tool written in C.

## Features
- [x] CPU usage monitoring
- [x] Memory tracking
- [x] Process viewer

## Goals
- Learn how Linux system information works using `/proc`
- Build a lightweight system monitoring tool
- Understand low-level system programming concepts

## Development Progress
- [x] Project initialized
- [x] CPU usage module
- [x] Memory monitor
- [x] Process viewer

## Tech Stack
- Language: C
- Platform: Linux
- Data source: `/proc`

## Future Plans
- Add real-time updates
- Improve output formatting (CLI UI)
- Possibly implement a `top`-like TUI interface

## How to test
use `gcc main.c system.c -o sys_alert` to display some examples
